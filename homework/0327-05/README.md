網誌系統功能統整
基本功能
- 文章管理 - 發布、編輯、刪除自己的文章
- 使用者系統 - 註冊、登入、登出
- 帳號列表 - /users 頁面顯示所有帳號密碼
排版功能
- 標題 - H1 大標題、H2 中標題、H3 小標題
- 文字樣式 - 粗體、斜體、底線
- 段落 - 段落、引用區塊、列表
- 圖片 - 本地上傳、網址插入
推薦系統
- 按讚 - 每個使用者對每篇文章只能按一次，再按取消
- 熱門推薦 - 右側固定顯示按讚數最高的文章
頁面與導航
- 網站圖標 - 左上角 Blog 標誌，點擊回首頁
- 使用者頁面 - 點擊作者名稱查看該使用者所有文章
- 個人專屬 - 登入後顯示歡迎訊息，可進入自己頁面
風格
- 暗色系 - 深藍紫漸變背景
- 響應式 - 支援不同螢幕尺寸
---
啟動方式
npm install
del blog.db
node server.js
